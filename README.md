## Installation

Add this line to your application's Gemfile:

    gem 'knife-vm'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install knife-vm

## Usage

knife vm server delete
knife vm create 
knife vm provision
knife vm delete
knife vm show NODE
